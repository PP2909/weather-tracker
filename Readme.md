# 🌤️ WeathEra — Weather App

**Assignment 2 | Interactive Frontend Development**
**Made by Pragya Sinha**

A simple weather app built using HTML, CSS & JavaScript. Type a city, get instant weather!

---

## ✨ What it does

- Search weather for any city in the world
- Shows temperature, humidity, wind, pressure & visibility
- 5-day forecast — all in one screen, no scrolling
- Works on mobile too!

---

## 🛠️ Built with

`HTML` · `CSS` · `JavaScript` · `OpenWeatherMap API`
